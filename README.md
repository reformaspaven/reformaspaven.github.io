# PAVEN REFORMAS

Website for PAVEN REFORMAS - Reformas, Impermeabilizaciones y Pintura Profesional.

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

This project automatically deploys to GitHub Pages on push to the `main` branch.
